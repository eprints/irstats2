IRStats2

Statistical tools for EPrints

Download this project as a .zip file Download this project as a tar.gz file

Information

IRStats2 is a statistical framework for EPrints - Not only it comes with default tools and reports but it also allows you to customise it and, for instance, add new metrics or data.

IRStats2 is developed against EPrints 3.3 but it was written to also work on EPrints 3.2. Older versions of EPrints are, however, not supported (and will never be).

Please read along to find out how to get it and how to customise it.
